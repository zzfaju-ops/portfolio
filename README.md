# portfolio
# Zoe Faju — Developer Portfolio

## Overview
A personal portfolio website built from scratch to showcase my projects, 
technical skills, and experience to recruiters and hiring managers. 
Designed to be visually professional, fully responsive, and easy to update.

## Live Site
https://zzfaju-ops.github.io/portfolio

## Features
- Responsive design that works on desktop and mobile
- Clickable project cards with a detail panel for each project
- Admin panel (password protected) to add and remove projects without touching code
- Smooth scroll navigation and animated sections
- Direct links to GitHub, LinkedIn, and email

## Tech Stack
- **Languages:** HTML5, CSS3, JavaScript
- **Fonts:** Google Fonts (Playfair Display, DM Sans)
- **Hosting:** GitHub Pages
- **Tools:** Git, Terminal, VS Code

## How to Run Locally
1. Clone the repo:
   `git clone https://github.com/zzfaju-ops/portfolio`
2. Navigate into the folder:
   `cd portfolio`
3. Open in your browser:
   `open index.html`

## What I Learned
- How to design and build a polished multi-section website from scratch
- Implementing a password-protected admin system using localStorage
- Writing clean, maintainable CSS with custom properties and variables
- Deploying a live site using GitHub Pages and Git version control
- How to structure a project for easy future updates
